Squares Type

Select what type of squares board you want. The default is a standard 100 square board. You can also choose a board that has only 25 squares where each square will be granted four numbers.

Admin Password

The admin password should not be shared with anyone else. It will allow you to administor your squares by changing certain settings, marking squares as paid, clearing squares, etc. Remember this password, because it cannot be recovered or reset.

Join Password

This is an optional password that you may use to password protect your squares.

Squares Unlock

No squares can be claimed prior to this date and time.

Squares Lock

Sets the date and time when squares can no longer be claimed.
